## 7. 公式速查

```
初始暴击率 = clamp(5% + WIS × 0.2%, 5%, 100%)
暴击伤害  = 1.5 + WIS × 0.05%
攻击间隔  = clamp(1.8 / (1 + SPD/100), 0.24s, 2.2s)
攻速(/s)  = 1 / 攻击间隔
伤害轨道  = ATK >= IAT×0.8 ? 外功(ATK vs DEF) : 内功(IAT vs IDR)
```
