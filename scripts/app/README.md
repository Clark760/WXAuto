# scripts/app

本目录承接应用编排层代码。

允许放入：

1. 战斗启动、阶段推进、商店刷新等用例协调逻辑
2. Coordinator、Presenter 装配代码
3. 场景和领域服务之间的桥接
4. 明确的用例入口，例如“开始战斗”“结算战斗”“刷新商店”“打开详情面板”

禁止放入：

1. 具体 UI 节点树声明
2. 纯领域规则实现
3. 根节点到处查找的临时胶水代码
4. 把旧 `battlefield` God Object 原样复制进来，只是换个目录名

当前阶段说明：

1. 本目录在 Phase 1 仍主要承担“冻结落点”角色。
2. 真正的 `BattlefieldCoordinator` / 应用编排迁移以 Phase 2 为主，不在本轮强行提前搬迁旧壳。
